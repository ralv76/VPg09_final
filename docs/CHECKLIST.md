# Чек-лист сдачи (ТЗ 7.3)

## Функциональные критерии (7.1)
- [ ] Пользователь может загрузить PDF и получить подкаст за 5–10 минут
- [ ] Сгенерированный подкаст содержит диалог двух разных голосов
- [ ] В подкасте присутствует фоновая музыка (регулируемая громкость)
- [ ] Создаётся RSS-лента, валидная для подкаст-платформ
- [ ] Интерфейс отображает прогресс генерации на всех этапах

## Технические критерии (7.2)
- [ ] Код размещён в публичном GitHub репозитории
- [ ] Проект развернут и доступен онлайн
- [ ] Документация включает инструкцию по запуску локально
- [ ] Приложение обрабатывает ошибки API с корректными сообщениями и рекомендациями
- [ ] Отсутствуют утечки памяти при длительной работе
- [ ] Автоматическое восстановление сервиса при перезагрузке сервера (systemd)

## Критерии сдачи курса (7.3)
- [ ] Предоставлено ТЗ (данный документ)
- [ ] GitHub репозиторий с исходным кодом
- [ ] 12+ скриншотов интерфейса
- [ ] 3–5 примеров готовых подкастов
- [ ] Демо-доступ к работающей платформе
- [ ] Видео-презентация 6–8 минут

## Запуск тестов
```powershell
.venv\Scripts\activate
pytest tests\ -v
```
